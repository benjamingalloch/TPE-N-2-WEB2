## Importar base de datos
- desde PHPMyAdmin (o cualquiera) database/db_movies.php

## Pueba con postman

(GET) Lista completa de Movies: http://localhost/ejercicios/TPEspecialApi/api/movies

(GET) Movie por ID: http://localhost/ejercicios/TPEspecialApi/api/movies/ID

(GET) Lista completa ordenada: http://localhost/ejercicios/TPEspecialApi/api/movies/ATRIBUTO/ASC - DESC

(POST) Insertar Pelicula: http://localhost/ejercicios/TPEspecialApi/api/movies 

(DELETE) Eliminar Movie por ID: http://localhost/ejercicios/TPEspecialApi/api/movies/ID

(PUT) Editar Movie http://localhost/ejercicios/TPEspecialApi/api/movies/ID
